# AI Support Agent Backend

A backend application built using Java and Spring Boot that demonstrates
agent-based architecture using an orchestrator, worker agents, and tools.

## Objective

- Learn backend architecture step by step
- Understand AI agents clearly (no black boxes)
- Build a real-world, interview-ready project

## Tech Stack

- Java
- Spring Boot
- Maven
- PostgreSQL (later)
- AI / LLM integration (later)

## Learning Philosophy

This project is built incrementally.
Every concept is explained before implementation.
